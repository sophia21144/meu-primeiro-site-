Tag p

A tag p define um parágrafo de texto. Ela é usada para organizar o conteúdo em blocos.

Tag mark

A tag mark destaca uma parte do texto como se estivesse marcada com um marca-texto.

Este texto está marcado.

Tag small

A tag small reduz o tamanho da fonte, normalmente usada para observações ou rodapés.

Tag i

A tag i deixa o texto em itálico, útil para palavras em destaque ou estrangeiras.

Tag u

A tag u sublinha o texto, mas hoje em dia é mais comum usar CSS para isso.

Tag strong

A tag strong dá ênfase forte ao texto, geralmente deixando-o em negrito com valor semântico.

Tags ol, ul e li

ol cria uma lista ordenada, ul cria uma lista não ordenada e li representa cada item da lista.

Lista ordenada:

  1. Primeiro item
  2. Segundo item

Lista não ordenada:

  • Item A
  • Item B

Tag hr

A tag hr insere uma linha horizontal para separar seções de conteúdo.

Tags sub e sup

A tag sub cria texto subscrito (abaixo da linha) e sup cria texto sobrescrito (acima da linha).

Fórmula: H2O | Exponenciação: 102 = 100

Tag iframe (vídeo do YouTube)

A tag iframe é usada para incorporar conteúdo externo dentro da página, como vídeos do YouTube, mapas do Google, ou outro site. Para vídeos, você deve usar o link no formato "embed" fornecido pelo YouTube.

Tags de tabela

A estrutura básica de uma tabela usa as seguintes tags:

  • table: cria a tabela
  • tr: define uma linha
  • th: define um cabeçalho de coluna
  • td: define uma célula comum
  • caption: título da tabela
  • thead, tbody, tfoot: seções da tabela

Exemplo:

Tabela de Alunos
Nome Idade
Maria 20
João 22
